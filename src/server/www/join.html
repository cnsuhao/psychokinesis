Psychokinesis

构筑便捷的资源分享网络

JavaScript
JavaScript如今已经日趋完善。现在不论是Web前端还是后台都可以使用JavaScript进行开发，而且性能并不逊色。Psychokinesis Web前端使用JQuery+AngularJS进行开发，后台接口也大量使用Node.js开发。
Erlang
Erlang在并行处理方面具有得天独厚的优势，使用它实现网络数据分发自然是不二之选，除此之外它在集群方面也颇有优势。Psychokinesis使用基于Erlang编写的ejabberd作为通讯枢纽，处理来自各个节点的数据。
DHT网络
P2P网络中数据分散在网络里的各个节点上，如何在众多分散的数据中快速地找到所需的资源对于建设一个有效的P2P网络至关重要。DHT为此提供了一种可行的方案，基于它我们便可充分挖掘P2P网络资源。
